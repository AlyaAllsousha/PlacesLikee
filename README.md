Социальное приложение для путешественников, позволяющее отмечать посещенные места на карте, делиться маршрутами и находить интересные локации от других пользователей.

---

##  Описание

**PlacesLikee** — это платформа для сообщества путешественников, где пользователи могут:
- Отмечать на карте места, которые они посетили
- Делиться впечатлениями и фотографиями
- Находить рекомендации от других путешественников
- Подписываться на авторов и получать уведомления о новых метках

---

##  Функционал

###  Для гостей
| Функция | Описание |
|---------|----------|
| ️ Просмотр карты | Отображение всех меток на карте Yandex MapKit |
|  Список меток | Просмотр всех меток в виде ленты |
|  Создание метки | Быстрое добавление метки без авторства |
|  Поиск | Поиск по автору, тексту описания, тегам |
|  Сортировка | По алфавиту авторов, по количеству лайков |
|  Фильтрация | Только с фото / без фото |
|  Оффлайн-режим | Просмотр кэшированных данных без интернета |

###  Для зарегистрированных пользователей
| Функция | Описание |
|---------|----------|
|  Личный профиль | История созданных меток, статистика |
|  Лайки | Возможность лайкать/убирать лайк у меток |
|  Авторство | Создание меток с привязкой к аккаунту |
|  Удаление | Удаление собственных меток |
|  Уведомления | Push-уведомления о новых метках от подписок |
